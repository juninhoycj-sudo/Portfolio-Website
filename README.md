# 🖥️ Dev Portfolio Template

A clean, techy personal portfolio template for software engineers. Dark theme, terminal aesthetic, no dependencies — just one HTML file.

![Preview](preview.png)

## ✨ Features

- Single `index.html` file — no build tools, no npm install
- Sections: Hero, About, Skills, Projects, Contact
- Responsive down to mobile
- Easy to customize — all editable areas are marked with `<!-- EDIT -->` comments

## 🚀 Deploy in 60 seconds

Click the button below to clone this template and deploy it instantly on Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOURUSERNAME/portfolio-template)

> Replace `YOURUSERNAME` in the URL above with your actual GitHub username after you've forked the repo.

Or deploy manually:

1. Fork this repo
2. Go to [vercel.com](https://vercel.com) → New Project → Import your fork
3. Hit Deploy — done

## 🛠️ How to customize

Open `index.html` and search for `<!-- EDIT -->` — every section that needs your info is marked.

| Section | What to change |
|---|---|
| `<title>` | Your name |
| Nav logo | Your handle or domain |
| Hero | Name, title, bio, terminal lines |
| About | Bio paragraphs, stats (years, projects, stars) |
| Skills | Add/remove tags per group, or rename groups |
| Projects | Name, description, stack tags, link, language dot color |
| Contact | Email, GitHub, LinkedIn, resume link |
